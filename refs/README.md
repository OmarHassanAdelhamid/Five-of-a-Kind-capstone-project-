# References (`refs/`)

This folder holds **bibliography and supporting materials** for AutoVox capstone documentation (papers, standards, and citation sources).

LaTeX documents under `docs/` typically use a shared `References.bib` (or project-specific `.bib` files) for BibTeX. A common naming convention for citation keys is **AuthorYear**; for two authors, **Author1Author2Year**; for three or more, **Author1EtAlYear** (last names).

Add new PDFs or notes here when they support requirements, design, or VnV traceability.
