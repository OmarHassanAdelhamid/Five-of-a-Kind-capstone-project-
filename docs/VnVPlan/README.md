# Verification and Validation Plan

The folders and files for this folder are as follows:

- ```README.md``` Short overview of this documentation
- ```Makefile``` Build configuration for Latex
- ```VnVPlan.tex``` Latex source code for the associated pdf
- ```VnVPlan.pdf``` Documentation generated from the associated Latex

This document expands on the verification and validation plans described in the SRS, as well as stating specific testing cases with traceability to each requirement identified.

Usage: Run ```make``` within this folder to regenerate documentation
