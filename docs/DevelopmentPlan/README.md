# Development Plan

The folders and files for this folder are as follows:

- ```README.md``` Short overview of this documentation
- ```Makefile``` Build configuration for Latex
- ```DevelopmentPlan.tex``` Latex source code for the associated pdf
- ```DevelopmentPlan.pdf``` Documentation generated from the associated Latex

This document states primary plans for team communication, roles, and general workflow. Plans for the initial proof of concept are also identified, as well as initial plans for technology to be utilized.

Usage: Run ```make``` within this folder to regenerate documentation.
