# Hazard Analysis

The folders and files for this folder are as follows:

- ```README.md``` Short overview of this documentation.
- ```Makefile``` Build configuration for Latex
- ```HazardAnalysis.tex``` Latex source code for the associated pdf
- ```HazardAnalysis.pdf``` Documentation generated from the associated Latex

This document includes a Failure Mode and Effect Analysis of AutoVox, as well as additional requirements generated during this process.

Usage: Run ```make``` within this folder to regenerate documentation.
