# Project Management

The files for this folder are as follows:

- `POC_Team_Contrib.pdf` - Team statistics (commits, meetings) as of the POC, as well as info about the POC presentation.
- `POC_Team_Contrib.tex` - LaTeX to build the respective pdf file.
- `Rev0_Team_Contrib.pdf` - Team statistics as of the informal Revision 0 presentation.
- `Rev0_Team_Contrib.tex` - LaTeX to build the respective pdf file.
- `Final_Team_Contrib.pdf` - Team statistics as of the formal Revision 1 presentation.
- `Final_Team_Contrib.tex` - LaTeX to build the respective pdf file.
