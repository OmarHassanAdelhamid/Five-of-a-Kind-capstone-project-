# Problem Statement and Goals

The folders and files for this folder are as follows:

- ```README.md``` Short overview of this documentation
- ```Makefile``` Build configuration for Latex
- ```ProblemStatement.tex``` Latex source code for the associated pdf
- ```ProblemStatement.pdf``` Documentation generated from the associated Latex

This document contains the overarching problem the project aims to solve, identifies stakeholders, states the key goals, and identifies planned extra documentation to be created.

Usage: Run ```make``` within this folder to regenerate documentation.
