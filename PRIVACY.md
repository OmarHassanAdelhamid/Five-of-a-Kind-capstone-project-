# Privacy statement

AutoVox runs **on your machine**. STL files, voxel projects, and exports are processed locally by the application and FastAPI backend. Storage locations for the packaged desktop app are described in [INSTALL.md](INSTALL.md). This project does not operate a cloud service or user accounts for the application itself.

If you use **GitHub** (for example to file issues), your activity is subject to [GitHub’s policies](https://docs.github.com/en/site-policy). Participation in project discussions is also governed by the [Contributor Covenant Code of Conduct](CodeOfConduct.md).

For course, lab, or institutional requirements beyond this document, follow your supervisor and organization’s rules.
