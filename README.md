# AutoVox

Developer Names: Omar Abdelhamid, Daniel Maurer, Andrew Bovbel, Olivia Reich, Khalid Farag

Supervisor: Dr. Onaizah

Date of project start: September 12, 2025

This project's goal is to reduce the manual effort involved in creating voxel-based multi-material 3D-print designs through the development of a specialized tool, allowing users to import and convert standard STL files to a voxel-based recreation.

The folders and files for this project are as follows:

docs - Documentation for the project
refs - Reference material used for the project, including papers
src - Source code
test - Test cases
